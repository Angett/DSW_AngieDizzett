Introducción

El contenido de HTML5 se conforma por una variedad de elementos, que adquiere un valor semántico, se cuenta con etiquetas que poseen un gran valor para diferenciar las secciones que se identifican en la página web, estas etiquetas son: header, nav, aside, article, section, footer. El correcto uso de estas etiquetas conforma una estructura deseada para el diseño de la página web.
